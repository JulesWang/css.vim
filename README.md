Vim CSS Syntax File
------------

Installation
-------------

## Manual

copy `css.vim` file into your `~/.vim/syntax/` directory;

## Using Vundle:

add the following line to your `~/.vimrc` file;

```
Bundle "JulesWang/css.vim"
```

and run `:BundleInstall` in vim.
